# Проект: Основы промпт-инжиниринга (prompt-basics)

Этот репозиторий содержит результаты исследования базовых принципов взаимодействия с большими языковыми моделями (LLM) на примере анализа кода из Piscine.

## Выводы исследования (Summary)

* Самая большая разница между размытым и точным запросом заключается в том, что размытый промпт заставляет модель давать поверхностные, обобщенные ответы с похвалой базовой логики, в то время как точный запрос с жесткими ограничениями (контекст, Big O, типы данных) мгновенно переводит ИИ в режим глубокого технического аудита с конкретными цифрами и граничными случаями.
* Понижать температуру модели необходимо в тех случаях, когда вам требуется получить строго детерминированный, предсказуемый и точный результат, полностью исключающий фантазию ИИ, например: при написании или дебаггинге программного кода, математических расчетах или форматировании данных по шаблону.
